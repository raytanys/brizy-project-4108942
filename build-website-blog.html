Deciding between Brizy Cloud vs Wordpress

Brizy Website Guide (EASIEST)
Wordpress Website Guide

Since I provide 2 Website Setup guides, I'll like you to make an informed decision on what to go for.

  1. Brizy Cloud (recommended) - Super easy, non-technical, you don't have to worry about hosting, site maintenance and upkeep.
  2. Wordpress - Need for hands on hosting, maintenance, troubleshoot plugin conflicts, website security. Open source and powers 1/4 of the world's websites.

I would compare using Brizy Cloud to using an iPad, and compare using Wordpress to assembling computer parts to set up your own computer.

It's a decision between "it just works" vs "full control".

Step #1 - Sign up for a Brizy Free Account

To get started, simply go to Brizy Cloud, and click Account Login.

Select the account icon on the top right, and click "Sign Up".

Sign up for an account with your email and password. At this point, you’ll notice that you only need your email address, and no credit card is required.

Once you're signed up, click "Create New Project".

Select "Website & Landing Page"

Your website project is created. Simply click into it to edit.

Getting started is really easy, isn't it?

Step #2 - Start Creating Content

The basic structure of Brizy can be understood via:

  1. Layouts - layouts are prebuilt pages that are themed consistently for various website designs (e.g. Business, Food, Health & Beauty) that Brizy has created, to help you get started in the shortest possible time. These have been professionally designed, and require very little time on your part. Everything is editable.
  2. Blocks - Blocks are building blocks within a page. Think of them as sections within your webpage. The great thing about Brizy blocks is that they are really pre-built, provide inspiration and much easier to get started.

1) Create Pages with Blocks

2) Create Entire Websites with Layouts

1) Create Pages with Blocks

Simply start by adding a block.

You can choose to start from a blank block, or use one of the elegantly designed pre-built blocks as inspiration.

Voila! You have a block to start working with.

Edit Text

To edit text, simply point your cursor at any text element and type away.

You can change typography, font colors, bold, italicize, etc.

Add Elements

Think of "Elements" as text, images, but also video, icons etc.

To add an element, simply click on the "plus" icon on the top left, and select an element.

Simply drag and drop the element onto the page.

Change Global Fonts

Changing fonts globally for consistency on your site is super easy.

Click on the "Brush" icon.

Select each font type (e.g. Heading 1, Heading 2). Change the font type, size, weight accordingly.

That's it! Your site will be updated globally throughout your website for these various font types.

Change Global Colors

Changing colors throughout your site is super easy as well.

Checking the global colors will update various parts of your website, like buttons, icon colors etc.

Rearranging Blocks

As you add blocks to your page, rearranging the order of the blocks is also super easy.

Simply select the "Drawer" icon on the left, and drag and drop the blocks to rearrange them on the page.

Headers & Footers (Pro)

To add Headers or Footers, add another block.

Select "Header" or "Footer" in the category section on the left.

Note that these are Brizy Pro features, and you'll need to upgrade to utilize them.

Saving or Publishing the Page

To save or publish the page, go to the bottom right corner, and choose to either "Save Draft" or "Publish Page" accordingly.

Back to Brizy Dashboard

To return back to the dashboard, click on the icon at the bottom left of the page.

2) Create Entire Websites with Layouts

The fastest way you can create entire websites with Brizy is via "layouts".

Instead of adding a block, select the layout tab.

You can scroll through all the layouts available, or filter by category.

Most layouts are Pro features, which you will need to upgrade, but it's totally worth it, as it allows you access to Pro features, Headers and Footers, Pop Ups and integrations, hosting and more.

To use the Pro layout, click on "Get Brizy PRO".

You can scroll through the layout pack to see the beautifully created pre-built pages that you can use once you have upgraded.

I have since upgraded to Brizy Pro and I can make use of these layouts.

Select the page. Make sure to click "Replace global styling" so all the fonts, colors that you see in the example will carry over to your website.

Then click "Import This Layout".

Just like that, the layout for that page is imported.

You can easily make edits to any part of the page as well.

I can easily add additional pages thereafter.

I chose the same layout pack, and selected the About page this time.

Once again, I can edit text on the page easily.

I can also replace images on the page easily, rearrange blocks, delete blocks or elements as I like.

Step #3: Decide and Purchase a Domain Name

So far, you have been putting together the elements into a website.

If you want to publish your site as "yoursitename.com", you’ll have to purchase your own domain name.

And that happens outside of Brizy.

I personally purchase and maintain ALL my domain names on Namecheap. I no longer use domain names hosted by other providers, as I find it best to consolidate all my domain names within Namecheap.

I like domain names ending with ".com". E.g. MyName.com or MyCompany.com. Other great options include ".org", or ".net".

To search for a domain, type into the search field below.

Click "Checkout".

Step #4: Publish Your Website to Your Domain

In your Brizy Cloud dashboard:

Click on "Publish".

The thing I love about Brizy, is even though they provide the technology to create your website, they do not hold you hostage unlike other providers like Squarespace, Wix and Weebly. They offer 4 options in publishing, and include an option to export to html.

For the purposes of this guide, I will focus on publishing your site to Brizy Cloud, hosted by Brizy.

To do so, click on "Custom Domain".

Brizy is super generous in providing 1 free (really free) domain hosted by them, entirely free. Did you say free? Yes it really is.

Type in the domain name you have purchased on Namecheap (or any other registrar).

You will be provide instructions on pointing your domain name to the website that you have created.

Now, let's go back to the Namecheap window.

Create 2 records, exactly as what it shows on your Brizy page.

  1. "A" record
  2. "CNAME" record

It may take couple of hours for records to be updated. In my experience, however, it can be as quick as 5 mins.

Creating Pop Ups in Brizy Cloud

Brizy Cloud is an amazing website builder, but also has great pop up / lead generation capabilities.

Why Create Pop Ups?

Popups are a really good way to offer website visitors additional information, tools, and continually engage them.

By offering something of value (e.g. a discount code, a free course) - you can continue to engage website visitors via channels like email even after they leave your page.

Make sure though to only be useful, relevant and never be spammy.

How to Create Pop Ups in Brizy?

Simply select from one of the pre-built popups or start from scratch.

Every element of the pop up is easily editable - text, images, buttons, forms.

Decide on the trigger conditions of the pop up. On page load, on page exit intent, etc..

Choose what pages the pop up should appear. All pages, select pages, and even exclude specific pages from displaying the pop up.

Frequently Asked Questions (FAQs)

Since I receive numerous questions on starting a website on Brizy Cloud, I've put together this FAQ section that can help you with the most immediate queries.

How much does setting up a website cost me?

There are 2 costs associated with setting up your website:

  1. Your domain name (typically $12/ year) or so if you go for a .com, .net, or .org
  2. Brizy Cloud subscription ($79/ year or $279 / year)

That's only $1/mth for your domain name, and $7/mth for your website!

Should I go for the free, $79/ year, or $279 / year option?

The great thing is you can get started immediately for free.

Sign up for a free Brizy Cloud account and play around with the website builder. You can create test sites like mytestsite.brizy.site. If you want to have it on your own domain, e.g. myname.com - you'll have to upgrade to the paid option.

The $79 / year option lets you host 1 website with Brizy Cloud, which should be more than sufficient if you only need 1 website to start with.

When you get hooked to building websites (it's really fun ;)), then you might want to upgrade to the $279/ year option to host 20 websites. It's up to you.
